# Task #52 — Implementation Plan

## Goal
Create `app/pages/12_rv_modeling.py`: statistical RV modeling page with improved two-component mixture model.

## Key Improvement Over Notebook
- **Notebook**: Both single and binary ΔRV distributions assumed Gaussian → 4 free params (f_bin, σ_s, σ_b, amp)
- **New**: Single stars use Gaussian range distribution; binary stars use **empirical** distribution from Monte-Carlo orbital simulation → 2 free params (f_bin, σ_s)

## Mathematical Model
```
f_obs(t) = (1 - f_bin) · S_single(t; σ_s) + f_bin · S_binary(t)
```
Where:
- `S_single(t; σ_s)` = P(Range > t | n Gaussians with std σ_s) — analytical, precomputed on standardized grid
- `S_binary(t)` = Empirical survival function from simulated binary ΔRVs (using `simulate_delta_rv_sample` with f_bin=1.0)

## Data Sources
1. **Observed ΔRVs**: `cached_load_observed_delta_rvs(settings_hash(settings))` from shared.py
2. **Binary ΔRVs**: Generated by `simulate_delta_rv_sample(f_bin=1.0, ...)` from `wr_bias_simulation.py`
3. **Single-star ranges**: Empirical via numpy simulation of Gaussian ranges (500K samples, fast)

## Architecture
1. Standard page header (sys.path, inject_theme, render_sidebar)
2. Cached functions for both survival distributions
3. Sidebar: simulation parameter controls + "Run Fit" button
4. Two-stage curve_fit (raw data → significance-filtered)
5. Four display panels + simulation details expander

## Sidebar Controls
- Period model (Dsilva/Langer)
- π or weight_A depending on model
- N_sim (10K–500K)
- n_epochs (2–20, default 6)
- Time span (days)
- Advanced: eccentricity model, mass-ratio model, seed

## Display Panels
1. **Fitted curve + observed**: f_obs(t) scatter with errorbars, model curve, single/binary components, optimal threshold
2. **Weighted PDFs**: (1-f)·PDF_single and f·PDF_binary with filled areas, crossover threshold
3. **Residuals**: normalized (obs-model)/σ with ±2σ bands, χ²_red annotation
4. **Parameters table**: f_bin ± err, σ_single ± err, χ²_red, Bartzakos-corrected f_global
